html中信息框的示范：
我们需要在head中创建一个function用来关闭。notification是物品id。
function closeNotification() {
    document.getElementById('notification').style.display = 'none';
}
然后再在body中创建信息框。在第二个span中填写信息，在第一个span中的onclick中填写function。把yellow替换成实际的背景颜色。
<div id="notification" style="background-color: yellow; padding: 10px; position: fixed; bottom: 0; left: 0; right: 0; display: flex; align-items: center; z-index: 1000;">
    <span onclick="closeNotification()" style="cursor: pointer; font-weight: bold; margin-right: 10px;">×</span>
    <span>在这里填写信息</span>
</div>
如果你想显示信息框，可以创建一个function，只需要把刚才的function中的none改为block
